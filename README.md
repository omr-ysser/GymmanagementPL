# Gym Management System

##  Overview

A web-based Gym Management System designed to streamline gym operations by managing members, subscriptions, and attendance. The system helps automate daily administrative tasks and improve overall efficiency.

---

##  Features

* Member registration and profile management
* Subscription plans and payment tracking
* Attendance tracking system
* Admin dashboard for managing operations
* Data storage and retrieval using SQL Server

---

##  Tech Stack

* **Backend:** ASP.NET MVC (.NET Framework)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** Microsoft SQL Server

---

##  How It Works

1. Admin registers members and assigns subscription plans
2. Members’ attendance is recorded daily
3. System tracks active/inactive subscriptions
4. Data is stored and managed efficiently in the database

---

##  System Benefits

* Reduces manual paperwork and administrative effort
* Improves accuracy in tracking members and subscriptions
* Enhances overall gym management efficiency
